implementing mathematical calculations only with binary operations.
used operations :
	&&, !=, ||, ==, !. =
	etc : sizeof, ., ,, [], *, &, ->, ...
incrementing or decrementing index variable is allowed.
	ex) i++, used like (num & (1 << i)) or in for/while loop
using <<. >>, &, ^, | is allowed when converting bool[] to int or reversed situation.
